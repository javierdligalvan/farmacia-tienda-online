# Contexto del Proyecto — Farmacia Muro

## Objetivo
Análisis de ventas (2024–feb.2026) de la farmacia recién adquirida. Meta: dashboard en **Power BI Desktop** (u otra herramienta gratuita eficiente) con KPIs e insights sobre fortalezas, debilidades y oportunidades. Fecha de traspaso farmacia: **28/02/2026**.

---

## Dataset Final (`PRODUCTOS_ALL_ENRICHED.csv`, <20k filas)
|---------------------|--------------------------------------------------------------------------|
| Columna             | Descripción                                                              |
|---------------------|--------------------------------------------------------------------------|
| `CODIGO`            | Código Nacional del medicamento o parafarmacia                           |
| `NOMBRE`            | Nombre del producto                                                      |
| `LABORATORIO`       | Fabricante/laboratorio                                                   |
| `CATEGORÍA`         | Medicamento: nivel ATC-4 (ej. `A01AA`). Parafarmacia: ver tabla inferior |
| `SUBCATEGORÍA`      | Medicamento: nivel ATC-3 (ej. `A01A`). Parafarmacia: ver tabla inferior  |
| `PVP`               | Precio de venta al público (referencia: 06/03/2026)                      |
| `AÑO`               | Año de las ventas (2024, 2025 o 2026)                                    |
| `ENERO`…`DICIEMBRE` | Unidades vendidas en cada mes del año `AÑO`                              |
|---------------------|--------------------------------------------------------------------------|
---

## Categorización

### Medicamentos
- Fuente: **VADEMECUM** (ATC)
- `SUBCATEGORÍA` = ATC nivel 3 (ej. `A01A`)
- `CATEGORÍA` = ATC nivel 4 (ej. `A01AA`)
